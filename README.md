# CatanGame

A Java implementation of the classic board game Catan.

## About

This project implements the strategic board game Catan, where players compete to build settlements, cities, and roads on an island while managing resources and trading with other players.

## Project Details

- **Language:** Java
- **Repository:** [AustinGitGreen/CatanGame](https://github.com/AustinGitGreen/CatanGame)
